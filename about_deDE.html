About

jPrinterAdmin ist reines Hobby-Projekt in der Entwicklung.

Es sollte in der Lage sein Daten von Druckern im lokalen Netzwerk via

- snmp und

- http

auszulesen und diese in einer Datenbank zu speichern. Da kannst wählen zwischen

- lokal (sqlite)

- MYSQL und

- Microsoft SQL-Server.

Man kann Berichte erstellen und diese ausgeben als

- HTML

- OpenDocument Spreadsheet (odt) oder

- Komma-getrennte-Werte (csv)

auf das loakle Datei-System oder an eine E-Mail-Adresse senden.

Man kann folgende Aufgaben in einem geplanten Task automatisieren:

- Auslesen der Druckerdaten

- Erstellen und Versenden von Berichten

- Datenbanksicherung exportieren



jprinterAdmin ist in Java mithilfe der Netbeans IDE geschrieben und unter der GNU Public License veröffentlicht.

Es verwendet folgen Komponenten von Drittanbietern und zum Teil angepassten Beispiel.Code.



snmp4j 2.1

ini4j 0.5.2

sqlitjdbc v0.56

jsoup 1.7.2

mysql-connector 5.1.7

jtds 1.2

jOpenDocument 1.3
